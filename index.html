Tools & Projects
~/tools/pycad
PyCAD
Probabilistic Analysis · Python/C++ · 2026
Computes exact output distributions of nonlinear functions over uncertain inputs using Cylindrical Algebraic Decomposition and Z3 — replacing Monte Carlo approximation with provable symbolic results. Includes automated K-S statistical validation. Built from scratch during my postdoc.
Python C++ Z3 QEPCAD B Maxima
github.com/hamzabandukara/pycad
~/tools/ram-c
RAM-C
Model Checker · Published at CSL '26
Checks whether infinite-alphabet state machines satisfy behavioural safety properties — implementing the novel modal logic I formulated in my PhD. The theory-to-tool pipeline: I designed the logic, proved its properties, and built the checker.
Python Java Haskell Model Checking
github.com/HamzaBandukara/RAM-C
~/tools/rabit
RABiT
Bisimulation Checker · SETTA '22 / JSA '23
On-the-fly bisimilarity checking for fresh-register automata — computes behavioural equivalence over infinite alphabets without full state-space exploration. Benchmarked against existing deterministic approaches with measurable performance improvements.
Java Python Bisimulation Benchmarking
github.com/HamzaBandukara/FRABisim
~/tools/paf
PAF
Modernised Fork · Floating-Point Error Analysis · 2025
Modernised fork of SOARLAB's probabilistic floating-point error analysis tool. Updated to Python 3.10+, latest Rust toolchain, Z3-exclusive solving, and added Docker support for cross-platform deployment. Original analysis by SOARLAB, University of Utah.
Python Rust Z3 Docker Floating-Point
github.com/HamzaBandukara/paf
